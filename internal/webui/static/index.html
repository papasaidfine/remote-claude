remote-claude

Your ~/.ssh/config hosts and their reverse tunnels. Editing a host writes its ssh config; “Start” just launches the tunnel per that config. Keep this window open to keep tunnels up.

Written as SetEnv LC_CLIENT_NAME on hosts you add; the server reaches you back as ssh <name>.

Local ssh server

The server reaches you back through the tunnel into this machine's sshd. Installing/hardening it may prompt for sudo in the launching terminal.

Xray

Optional, for censored/slow networks. Download the xray binary, then add vless:// nodes; turn on “xray” per host to route it through them.

Nodes — one vless:// URL per line; each connection picks a random node.
